
# 08 - Área do Cliente

## Visão Geral

A **Área do Cliente** é o espaço onde os clientes finais acessam suas informações, garantias, documentos, programa de fidelidade e mais.

## Integrações Futuras

Todas as páginas da área do cliente serão integradas com o Supabase para:
- Carregar dados reais do cliente
- Carregar configurações white label via Runtime
- Gerenciar autenticação
- Carregar garantias, documentos, timeline, etc.
